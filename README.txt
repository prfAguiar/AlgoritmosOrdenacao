Como rodar o programa:
Abra uma IDE ou um editor de código de sua preferência ou utilize o prompt de comando no diretório onde estão os arquivos java.

Para utilizar em um terminal:
Compile o arquivo utiizando o comando: "javac index.java". 
Rode o arquivo utilizando o comando "java index".

O programa funciona com menus para uma melhor interação com o usuário. Dentro dos menus o usuário escolherá o método de ordenação que queira 
e em seguida escolherá o arquivo que será ordenado. Realizando essas etapas, será criado um arquivo .csv no diretório onde estão os arquivos java com todos os 
dados ordenados e no final dele o número de movimentações realizadas. O usuário terá total liberdade para decidir se deseja ordenar novamente ou se deseja sair do programa.
